Tourism Management Program
> This project was created for a university coursework in Computer Science, focusing on C++ programming and system design.

## Description
This C++ program helps users plan and book trips based on their preferences and budget. It allows:
- Destination selection
- Counting passengers (adults and children)
- Budget and duration input
- Selection of hotels, restaurants, and transportation
- Display of best airline and activity suggestions
- Booking confirmation and file saving
